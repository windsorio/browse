This extension adds support for the Browse language to VSCode

## Features

- Syntax Highlighting
- Formatter
- Toggle Comments
- Code Folding
- Match, auto-close and auto-surround brackets
- Auto-close and auto-surround strings
- Auto Indentation

### Coming Soon

- Auto-Complete
- Code Navigation (jump to definition, etc.)
- Hover information
- Error checking
- Refactoring
